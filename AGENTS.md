# Regras do Projeto NUVA

- **Validação de Alterações ("oKei")**: Sempre que o utilizador enviar um "oKei" (ou variações como "Okey", "okei"), significa que a funcionalidade/layout atual está aprovada. Deves reter e proteger essa implementação exatamente como está, sem efetuar alterações nela, a menos que o utilizador dê uma ordem direta e específica no futuro para modificar essa parte.
